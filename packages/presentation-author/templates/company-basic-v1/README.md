# Company Basic v1 — Brand Frame Template

This directory contains a lightweight company brand frame template.

## Assets

- `manifest.json` — BrandFrame definition (slide size, header/footer placement, safe area)
- `header.png` — Placeholder header image (replace with real company asset)
- `footer.png` — Placeholder footer image (replace with real company asset)

## Usage

The header and footer images are automatically placed on every generated slide.
Body content is constrained to the safe area defined in the manifest.

## Replacing with Real Assets

1. Replace `header.png` with your company header (recommended: 13.333in × 0.45in at 150 DPI → 2000 × 68 px)
2. Replace `footer.png` with your company footer (same dimensions)
3. Adjust manifest coordinates if needed after visual inspection
