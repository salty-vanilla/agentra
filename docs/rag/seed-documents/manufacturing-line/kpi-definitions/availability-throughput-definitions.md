# 可動率・スループット KPI定義書

## 概要

第4製造ライン（line-4）で使用する主要KPIの定義・計算式・正常範囲・MESダッシュボード上の項目名との対応をまとめたドキュメント。

---

## KPI一覧

| KPI名 | 英語名 | MES項目名 | 単位 |
|---|---|---|---|
| 可動率 | Availability | `kpi.line4.availability` | % |
| スループット | Throughput | `kpi.line4.throughput` | pcs/h |
| サイクルタイム | Cycle Time | `kpi.line4.cycle_time` | sec/pcs |
| 設備総合効率 | OEE (Overall Equipment Effectiveness) | `kpi.line4.oee` | % |

---

## 可動率（Availability）

### 定義

計画稼働時間に対して設備が実際に稼働していた時間の割合。停止時間（故障・段取り・計画外停止）を差し引いた値。

### 計算式

```
可動率 (%) = (計画稼働時間 − 停止時間) / 計画稼働時間 × 100
```

### 用語定義

| 用語 | 説明 |
|---|---|
| 計画稼働時間 | シフト時間から計画休止（昼休憩・定期清掃等）を除いた時間 |
| 停止時間 | 故障停止・段取り替え・資材待ち・品質確認待ちの合計時間 |

### 正常範囲・目標値

| 区分 | 値 |
|---|---|
| 目標値 | ≥ 90% |
| 注意レベル | 80〜89% |
| 警告レベル | < 80% |

### MESでの確認方法

- MESダッシュボード → 「ライン稼働サマリー」タブ → `kpi.line4.availability`
- 時間軸: 日次・週次・月次で集計可能
- 停止時間の内訳は「停止理由別分析」で確認できる

---

## スループット（Throughput）

### 定義

単位時間あたりに生産された良品の数量。不良品・手直し品は含まない。

### 計算式

```
スループット (pcs/h) = 良品生産数 / 稼働時間 (h)
```

### 正常範囲・目標値

| 区分 | 値 |
|---|---|
| 目標値 | ≥ 120 pcs/h |
| 注意レベル | 100〜119 pcs/h |
| 警告レベル | < 100 pcs/h |

### MESでの確認方法

- MESダッシュボード → 「生産実績」タブ → `kpi.line4.throughput`
- リアルタイム値（直近1時間）と日次集計値の両方を表示

---

## サイクルタイム（Cycle Time）

### 定義

1個の製品を生産するのに要する時間。M1プレス機の1ストロークにかかる時間が基準。

### 計算式

```
サイクルタイム (sec/pcs) = 稼働時間 (sec) / 生産数
```

### 正常範囲・目標値

| 区分 | 値 |
|---|---|
| 目標値（定格） | 30 sec/pcs |
| 許容範囲 | 28〜35 sec/pcs |
| 警告レベル | > 35 sec/pcs または < 25 sec/pcs |

サイクルタイムが定格より大幅に短い場合（< 25秒）は過負荷運転の可能性があり、温度異常の原因となる。

---

## 設備総合効率（OEE）

### 定義

可動率・性能効率・良品率の積で表される、設備の総合的な使用効率。製造業における代表的な設備管理KPI（Overall Equipment Effectiveness）。

### 計算式

```
OEE (%) = 可動率 (%) × 性能効率 (%) × 良品率 (%)
```

| 構成要素 | 計算式 |
|---|---|
| 可動率 | （前述の定義に同じ） |
| 性能効率 | (実際のサイクルタイム / 定格サイクルタイム) × 稼働率 |
| 良品率 | 良品数 / 生産総数 × 100 |

### 正常範囲・目標値

| 区分 | 値 |
|---|---|
| 世界水準 | ≥ 85% |
| 目標値 | ≥ 80% |
| 注意レベル | 70〜79% |
| 警告レベル | < 70% |

### MESでの確認方法

- MESダッシュボード → 「OEEサマリー」タブ → `kpi.line4.oee`
- 構成要素（可動率・性能効率・良品率）の内訳もあわせて表示される

## KPI低下時の初動

| KPI | 低下時の主な原因 | 参照ドキュメント |
|---|---|---|
| 可動率低下 | 故障停止（E-TEMP-001 / E-VIB-002 等） | `runbooks/line-stop-initial-response.md` |
| スループット低下 | 設備パフォーマンス低下・段取り時間増加 | 保全担当へ報告 |
| サイクルタイム異常 | 設備過負荷 / 機械的抵抗増加 | `procedures/` 内の該当手順書 |
| OEE低下 | 上記の複合要因 | 製造主任・生産管理部門へ報告 |

---

## 注意事項

- MESのKPI値は稼働日の翌営業日に確定値として更新される。当日の値はリアルタイム速報値であり、若干の変動がある。
- 良品判定はライン末端の検査工程（外観検査・寸法検査）の結果を反映する。検査機器のキャリブレーション状態によって良品率に影響が出ることがある。
- KPIの定義・計算ロジックを変更する場合は生産管理部門・IT部門の承認が必要。
