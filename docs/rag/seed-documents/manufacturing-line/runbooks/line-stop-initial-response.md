# ライン停止初動ランブック

## 概要

第4製造ライン（line-4）でライン停止が発生した際の初動対応手順。  
エラーの種別によらず、停止後30分以内に実施すべき行動を定義する。

## 対象ライン

- 第4製造ライン（line-4）
- 主要機器: M1プレス機、M2搬送コンベア

## 役割分担

| 役割 | 担当 | 初動での主な責任 |
|---|---|---|
| オペレーター | 現場担当者 | 異常の発見・初期確認・保全担当への連絡 |
| 保全担当 | 設備保全チーム | 現地診断・応急処置・復旧判断 |
| 製造主任 | ライン責任者 | 影響範囲の判断・生産調整・上位報告 |

---

## 初動タイムライン

### T+0: 停止直後（オペレーター）

1. ラインが安全に停止していることを確認する（緊急停止ボタンが必要な場合は押す）
2. 操作パネルのエラーコードを確認し、記録する
3. 停止時刻・エラーコードをMES画面に入力する
4. M1・M2周辺の安全確認（煙・異臭・漏液・異音）を実施する
5. 異常が視認された場合は直ちに製造主任へ連絡し、周辺エリアの立入を制限する

### T+5分: 保全担当への連絡（オペレーター）

1. 保全担当へ以下の情報を伝えて連絡する
   - 停止時刻
   - エラーコード（例: `E-TEMP-001`、`E-VIB-002`）
   - 停止直前の稼働状況（異音・異臭の有無）
   - 目視で確認できた異常（煙・漏液等）

### T+10分: 現地到着・診断開始（保全担当）

1. 現地に到着し、オペレーターからの状況報告を聞く
2. 該当エラーコードのトラブルシューティング手順書を参照して診断を開始する
   - 温度異常 → `procedures/temperature-anomaly-troubleshooting.md`
   - 振動異常 → `procedures/vibration-anomaly-troubleshooting.md`
3. 応急処置が可能な場合は実施し、製造主任へ現状と見込み時間を報告する

### T+15分: 影響範囲の判断（製造主任）

1. 保全担当からの報告を受けて復旧見込み時間を確認する
2. 以下の判断基準に基づき対応方針を決定する

| 復旧見込み | 対応方針 |
|---|---|
| 30分以内 | 当該ラインの停止を継続。後続工程への在庫で吸収 |
| 30分〜2時間 | 後続工程の生産調整を開始。生産管理へ報告 |
| 2時間超 / 不明 | 設備管理部門・生産管理部門・上位管理職へ報告。代替対応を検討 |

3. 生産計画への影響が確定した場合は生産管理部門へ報告する

### T+30分: 状況確認（全員）

1. 保全担当は診断・処置の進捗を製造主任へ報告する
2. 原因未特定の場合はエスカレーション基準を確認し、設備管理部門へ支援要請を検討する
3. MESの作業日誌に現時点での状況を記録する

---

## 復旧確認チェックリスト

ライン再起動前に以下をすべて確認すること:

- [ ] エラーの原因が特定・修正されている
- [ ] 操作パネルのエラーアラームがクリアされている
- [ ] 安全カバー・パネルがすべて閉じられている
- [ ] 周辺エリアに作業者・工具・異物が残っていない
- [ ] 保全担当が再起動に立ち会っている（CRITICALレベル停止の場合）

---

## エスカレーション連絡先

| エスカレーション先 | 条件 |
|---|---|
| 設備管理部門 | 原因不明・修理に専門技術が必要・部品手配が必要 |
| 生産管理部門 | 復旧見込み30分超で生産計画への影響が発生する場合 |
| 上位管理職 | 2時間超の停止・安全事故・設備の重大損傷 |

## 注意事項・安全上のポイント

- 停止中に現場へ立ち入る場合は必ず保全担当の許可を得ること。
- 電気系統の点検は電気工事士有資格者のみが実施すること。
- 高温・高圧・回転体が関わる作業は該当の安全手順書を必ず参照すること。
- 復旧後の再発確認（5分間の監視）を省略しないこと。
