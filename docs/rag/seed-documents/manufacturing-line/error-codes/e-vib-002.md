# E-VIB-002: 振動異常エラー

## 概要

M1プレス機の主軸または駆動系で異常振動が検出された際に発生するエラーコード。  
ベアリング摩耗・アンバランス・ボルト緩みなどが主な原因として想定される。

| レベル | 条件 | システム動作 |
|---|---|---|
| WARN | 振動加速度 ≥ 4.5 mm/s² | アラームランプ点灯・オペレーターへ通知 |
| CRITICAL | 振動加速度 ≥ 7.0 mm/s² | ライン自動停止・保全担当へ緊急通知 |

## 対象設備

- ライン: 第4製造ライン（line-4）
- 機器: M1プレス機（主軸ベアリング部・駆動モーター取付部）
- センサー設置箇所: 主軸前後ベアリングハウジング・モーターブラケット

## 症状・信号

- 操作パネルに `E-VIB-002` 表示
- 異音（金属音・周期的なノック音）が聞こえる場合がある
- 制御盤内の振動警告ランプ（橙 / 赤）点灯
- MES画面にアラートログ出力
- WARN継続中は振動加速度のトレンドグラフが緩やかに上昇することが多い

## 推奨アクション

### WARNレベル（4.5 mm/s² ≤ 振動 < 7.0 mm/s²）

1. 操作パネルのアラームをミュートし、発生時刻と振動値をログに記録する
2. M1の稼働音を聴音し、金属音・周期的ノック音の有無を確認する
3. M1の外観点検を実施し、ボルト類の緩み・取付部のガタツキを確認する
4. 振動値が上昇傾向の場合は保全担当へ報告し、詳細点検の日程を調整する
5. 詳細診断は `procedures/vibration-anomaly-troubleshooting.md` を参照する

### CRITICALレベル（振動 ≥ 7.0 mm/s²）

1. ラインが自動停止していることを確認する（自動停止しない場合は緊急停止ボタンを押す）
2. M1周辺への立入を制限する（回転体・飛散物リスクあり）
3. 保全担当および製造主任へ即時連絡する
4. `runbooks/line-stop-initial-response.md` に従い初動対応を開始する
5. 原因特定（ベアリング・ボルト等の点検）まで再起動しない

## エスカレーション条件

以下のいずれかに該当する場合は、製造主任・設備管理部門へエスカレーションする:

- CRITICALレベルが繰り返し（同日2回以上）発生した
- 目視点検でベアリングの損傷・変色・過熱痕が確認された
- ボルト締め直し後も振動値が改善しない
- 原因不明のまま30分以内に復旧しない

## 注意事項・安全上のポイント

- 稼働中の回転部への接触は厳禁。点検は必ずM1停止・主電源OFFのうえ実施すること。
- ベアリング交換作業は保全担当有資格者が実施すること。
- 交換部品の記録（品番・交換日・作業者）を `maintenance/equipment-m1-maintenance-notes.md` に残すこと。
- 振動異常は早期に対処しないと二次損傷（シャフト損傷・歯車破損）につながるため、WARNレベルでも軽視しないこと。
