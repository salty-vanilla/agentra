# E-TEMP-001: 温度異常エラー

## 概要

M1プレス機の稼働温度が規定範囲を超えた際に発生するエラーコード。  
警告レベル（WARN）と緊急停止レベル（CRITICAL）の2段階で通知される。

| レベル | 条件 | システム動作 |
|---|---|---|
| WARN | 稼働温度 ≥ 75°C | アラームランプ点灯・オペレーターへ通知 |
| CRITICAL | 稼働温度 ≥ 90°C | ライン自動停止・保全担当へ緊急通知 |

## 対象設備

- ライン: 第4製造ライン（line-4）
- 機器: M1プレス機
- センサー設置箇所: 主軸ベアリング部・油圧ユニット・制御盤内

## 症状・信号

- 操作パネルに `E-TEMP-001` 表示
- 警告ブザー断続鳴動（WARN）または連続鳴動（CRITICAL）
- 制御盤内の温度警告ランプ（橙 / 赤）点灯
- MES画面にアラートログ出力

## 推奨アクション

### WARNレベル（75°C ≤ 温度 < 90°C）

1. 操作パネルのアラームをミュートし、発生時刻をログに記録する
2. M1周辺の換気口・冷却ファンの動作を目視確認する
3. 油圧ユニットの冷却水流量を確認する（流量計の目盛りが緑帯域内であること）
4. 5分後に温度が低下傾向にあれば監視継続、上昇傾向なら保全担当へ報告する
5. 詳細診断は `procedures/temperature-anomaly-troubleshooting.md` を参照する

### CRITICALレベル（温度 ≥ 90°C）

1. ラインが自動停止していることを確認する（自動停止しない場合は緊急停止ボタンを押す）
2. M1周辺への立入を制限し、安全距離（1m以上）を確保する
3. 保全担当および製造主任へ即時連絡する
4. `runbooks/line-stop-initial-response.md` に従い初動対応を開始する
5. 原因特定まで再起動しない

## エスカレーション条件

以下のいずれかに該当する場合は、製造主任・設備管理部門へエスカレーションする:

- CRITICALレベルが繰り返し（同日2回以上）発生した
- 冷却系統に異常（冷却水漏れ・ファン停止）が確認された
- 原因不明のまま30分以内に復旧しない

## 注意事項・安全上のポイント

- 高温部位への素手での接触は厳禁。耐熱グローブを使用すること。
- 消火器の設置場所は `safety/high-temperature-response-safety.md` を参照すること。
- 温度低下後も表面温度が下がるまで（目安: 40°C以下）点検作業を開始しないこと。
- 冷却水に腐食抑制剤が規定濃度で添加されているか、月次点検で確認すること。
